# HR Analytics Dashboard

A data-driven dashboard to analyze workforce trends — attrition, performance, demographics, and more — empowering HR teams to make smarter, evidence-based decisions.

## Overview
The HR Analytics Dashboard transforms raw employee data into actionable insights through interactive visualizations and structured reports. Built for HR professionals and business analysts, it simplifies complex workforce data into clear, digestible metrics.

## Features
- Employee Attrition Analysis — Identify patterns and risk factors behind turnover  
- Department-wise Insights — Drill down into performance and headcount by team  
- Gender & Age Distribution — Visualize workforce diversity at a glance  
- Salary & Performance Analysis — Correlate compensation with output  
- Interactive Filters (Slicers) — Slice data dynamically for targeted exploration  
- Visual Dashboard — Clean, intuitive charts and KPI cards  

## Tools & Technologies
- Power BI / Excel — Dashboard & Visualization  
- SQL — Data Querying  
- MS Excel — Data Cleaning & Preprocessing

## Dashboard Insights
- Attrition Hotspots — Flags departments with the highest employee turnover  
- Demographics Overview — Breaks down workforce by age, gender, and tenure  
- Performance Trends — Tracks individual and team-level performance over time  
- HR Decision Support — Surfaces actionable metrics for strategic workforce planning  

## How to Use
1. Download the dataset from the `data/` folder  
2. Open the dashboard file in Power BI or Excel (`dashboard/` folder)  
3. Apply filters using the provided slicers to explore specific segments  
4. Analyze the visual insights and export reports as needed
